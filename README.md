# haskell-tpb-itb

kumpulan tugas haskell ITB (tugas edunex) tahun ajaran 2023/2024

## how to compile haskell

- run ghci (download haskell first)
  `ghci`

- load the file first
  `:l [filename].hs`
  or
  `:load [filename].hs`, example `:l IsDateValid.hs`

- upon changes reload whats loaded currently
  `:reload` or `:r`

- exit ghci
  `:q`
